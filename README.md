# QQHomeworkBatchTool
爬虫批量下载QQ中的作业

> 前提：使用者需要有该群聊的管理者权利，才可以下所有人的作业

> 下载结果：该博文可以实现将群内所有作业全部下载下来放在`downloaded`，并按照每个实验单独一个文件夹、每个实验内按照n个成员生成n个对于文件夹（作业放在该文件夹中）
